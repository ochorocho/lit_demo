typo3-backend-color-picker

Wraps an <input type="text"> with the Alwan color picker. Supports preset swatches and an optional opacity slider.

typo3-backend-datetime

Display-only formatter — renders a semantic <time> element from an ISO date / Unix timestamp. Modes: absolute (default) and relative (auto-updates "5 minutes ago" → "6 minutes ago" every minute). The format attribute accepts date, datetime, or a Luxon-style pattern. This is not an input — for an interactive picker see typo3-formengine-element-datetime below.

typo3-formengine-element-datetime

The interactive datetime picker used by the FormEngine. Wraps the standard form-engine markup (.form-control-wrap > .form-wizards-wrap > .input-group) and initializes the flatpickr picker on the inner <input>. Identifies the input via the recordFieldId attribute (must match the input's id). Supports data-date-type values datetime, date, time, year, timesec, datetimesec.

typo3-backend-formengine-char-counter

Live character counter for an input or textarea. Bind it to a target field via the target attribute.

typo3-formengine-table-wizard

Interactive table editor used by the FormEngine "table" wizard. Reads its initial data from an external <textarea> referenced by the selector attribute (rows separated by newlines, columns by the configured delimiter).
