typo3-breadcrumb

Responsive breadcrumb. Auto-collapses overflowing nodes into a dropdown when the container is too narrow.

typo3-backend-tab-scroller

Wraps a Bootstrap-style tab list (<ul class="nav nav-tabs" role="tablist">) and adds horizontal scroll buttons when the tabs overflow. Tab activation is handled by @typo3/backend/tab — the data-typo3-tab attribute on each link points to the corresponding .tab-pane.
